# toeslagen-cal

Site which generates an ICAL containing payment dates for Dutch benefits ("toeslagen")

## Development

To run this repo locally, do the following after initially cloning the repo
```bash
pnpm install
```

And to run the development server do the following
```bash
pnpm dev
```

The site will then be served at [http://localhost:5173/](http://localhost:5173/)
